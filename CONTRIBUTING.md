# Contributing to OpenSoul

感谢你对 OpenSoul 的关注！

## 快速链接

- **GitHub:** https://github.com/NJX-njx/opensoul

## 如何贡献

1. **Bug 修复 & 小改进** → 直接提 PR！
2. **新功能** → 先开 Issue 讨论，然后提 PR。
3. **文档改进** → 欢迎提 PR。

## 开发流程

1. Fork 本仓库
2. 创建你的功能分支 (`git checkout -b feature/your-feature`)
3. 提交修改 (`git commit -m 'Add some feature'`)
4. 推送到分支 (`git push origin feature/your-feature`)
5. 提交 Pull Request

## 代码规范

- 提交前运行 `pnpm check`
- 确保测试通过 `pnpm test`
